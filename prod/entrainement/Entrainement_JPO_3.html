Exercice 7

On considère l'algorithme nb_occ qui compte le nombre d'apparitions d'une lettre dans un mot. Reconstituez la séquence d'instructions dans le bon ordre.

algo : nb_occ

paramètre : mot (str) : un mot.
lettre (str) : une lettre.

résultat : (entier) : le nombre d'apparitions de la lettre.

début :

Correction

La réponse correcte est:
  • Mettre le compteur à 0
  • Pour chaque caractère dans mot
  • Si le caractère est la lettre recherchée
  • Ajouter 1 au compteur
  • Retourner le compteur
