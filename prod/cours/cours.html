  • 1. Qu'est-ce qu'un type ?

    En Python il existe différents types de valeurs :

    Les entiers relatifs sont nommés: int
    15 ou -3 sont des int.

    Les réels sont nommés: float
    15.5 est un float.

    Les booléens ont deux états différents, True ou False (Vrai ou Faux) et sont nommés: bool
    16 < 18 est un bool à True.

    Les chaînes de caractères sont quant à elles nommées: str
    'Programmation' et 'c' sont des str. Respectivement une chaîne de caractère et un caractère.

    list: Une liste est déclarée par une série de valeurs séparées par des virgules, le tout encadré par des crochets. Elle est mutable, c'est-à-dire qu’on peut modifier ou supprimer les éléments qu’elle contient.
    [15, 76, 89, 'ordinateur'] est une liste.

    tuple: Un tuple est une séquence non modifiable de valeurs (immutable).
    ('apple', 'pomme') est un tuple.




  • 2. Les opérateurs (or, and, *, +, /)

    On retrouve les opérateurs dans des expressions qui fournissent une valeur résultant d’un calcul.

    Une expression a un type et une valeur.
    Comme en mathématiques, les expressions sont évaluées suivant les priorités des opérateurs et des parenthèses :

    1. or : "expression or expression" si une des expression est vraie alors la condition est vraie:
    True or False: True est vraie, alors la condition est vraie.

    2. and : "expression and expression" il faut que les deux soient vraies pour que la condition soit vraie:
    True and False: False n'est pas vraie, la condition est donc fausse.

    3. == et != : Égalité, Différent

    4. + et - : Addition, Soustraction

    5. * et / : Multiplication, Division

  • 3. Qu'est-ce qu'un Algorithme ?

    C’est un énoncé d’une suite d’opérations permettant de donner systématiquement la réponse à un problème.

    Les opérations et instructions d'un algorithme peuvent être rédigées de différentes manières telles que le français, et sont par la suite traduites dans un langage de programmation pour constituer un code.

  • 4. Qu'est-ce qu'une fonction ?

    Une fonction est une manière d'organiser et de réutiliser du code en le regroupant sous un nom spécifique.

    Elle prend des arguments en entrée, effectue des opérations, et renvoie un résultat.

    En Python il existe des fonctions déjà créées telles que "sort()" ou encore "min()" et "max()".

  • 5. Qu'est-ce qu'une assertion ?

    En programmation, l'instruction "assert" est utilisée pour vérifier qu'une condition spécifiée est vraie.

    Si la condition est vraie, le programme continue normalement.

    Si la condition est fausse, le programme lève une exception, généralement une "assertion error", indiquant qu'il y a un problème.

    L’instruction assert permet donc de vérifier qu’un code fonctionne

  • 6. La fonction len()

    La fonction len() en programmation est utilisée pour obtenir la longueur d'une séquence, telle qu'une chaîne de caractères, une liste, ou un tuple.

    Elle retourne le nombre d'éléments présents dans la séquence.

    len("paille") == 6

    len("paille"*2) == 12

    len("paille")*2 == 12

    len(82) → Erreur

    len(["toto", "tata", "paille"]) == 3

  • 7. La fonction range()

    La fonction range() en programmation est utilisée pour générer une séquence d'entiers dans un intervalle spécifié. Elle est souvent utilisée dans les "boucles for" pour itérer sur une séquence d'entiers.

    En Python, la fonction range() peut être utilisée de plusieurs manières, mais la forme la plus courante est range(a), où "a" est la valeur jusqu'à laquelle la séquence doit aller (non incluse).

    Par exemple, range(5) générera la séquence d'entiers de 0 à 4.

    Une autre forme courante est range(a, b), où "a" est la valeur de départ de la séquence et" b" la valeur jusqu'à laquelle la séquence doit aller (non incluse).

    Par exemple, range(2, 8) générera la séquence d'entiers de 2 à 7.
