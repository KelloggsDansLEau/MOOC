  • 1. Qu'est-ce qu'un type de valeur ?

    En Python il existe différents types de valeurs.
    Les entiers relatifs sont nommés int :
    15 ou -3 sont des int.

    Les réels sont nommés float :
    15.5 est un float.

    Les booléens ont deux valeurs, True ou False (Vrai ou Faux) et sont nommés bool :
    16 < 18 est un bool à True.

    Les chaînes de caractères sont quant à elles nommées str :
    'Programmation', "toto" et 'c' sont des str, respectivement deux chaînes de caractères et un caractère.

    Une liste est déclarée par une série de valeurs séparées par des virgules, le tout encadré par des crochets. Elle est mutable, c'est-à-dire qu’on peut modifier ou supprimer les éléments qu’elle contient et est nommée list :
    [15, 76, 89, 'ordinateur'] est une list.

    Un tuple est une séquence non modifiable de valeurs (immutable) :
    ('apple', 'pomme') est un tuple.




  • 2. Les opérateurs (or, and, *, +, /)

    On retrouve les opérateurs dans des expressions qui fournissent une valeur résultant d’un calcul.

    Une expression a un type et une valeur.
    Comme en mathématiques, les expressions sont évaluées suivant les priorités des opérateurs et des parenthèses :

    1. or : "expression or expression" si une des expression est vraie alors la condition est vraie :
    True or False : True est vraie, alors la condition est vraie.

    2. and : "expression and expression" il faut que les deux soient vraies pour que la condition soit vraie :
    True and False : False n'est pas vraie, la condition est donc fausse.

    3. == et != : Égalité, Différence

    4. + et - : Addition, Soustraction

    5. * et / : Multiplication, Division

  • 3. Qu'est-ce qu'un algorithme ?

    C’est un énoncé d’une suite d’opérations permettant de donner systématiquement la réponse à un problème.

    Les opérations et instructions d'un algorithme peuvent être rédigées de différentes manières telles que le français, et sont par la suite traduites dans un langage de programmation pour constituer un code.

  • 4. Qu'est-ce qu'une fonction ?

    Une fonction est une suite d'instructions qui définissent un sous-programme et qui renvoient un résultat pouvant être utilisé autant de fois que nécessaire dans un programme plus général.

    En Python, il existe des fonctions déjà existantes telles que sorted() ou encore min() et max(), mais il est possible de définir ses propres fonctions.

  • 5. Qu'est-ce qu'une assertion ?

    Une assertion est une proposition présentée comme vraie. Ne contenant aucune preuve de sa véracité, une assertion est potentiellement fausse. L'instruction assert est utilisée pour tester qu’une fonction renvoie le résultat attendu.
    Cependant, elle ne permet pas de s'assurer qu'une fonction renvoie le bon résultat dans tous les cas possible, il faudrait un nombre de tests trop conséquent.

  • 6. La fonction len()

    La fonction len() est utilisée pour obtenir la longueur d'une structure de données telle qu'une chaîne de caractères, une liste, ou un tuple. Elle retourne le nombre d'éléments présents dans la structure de données.

    len('paille') == 6

    len('paille' * 2) == 12
    En Python, il est possible de multiplier une chaîne de caractère par un entier, ce qui va concaténer la chaîne:
    'paille' * 2 == 'paillepaille'

    len('paille') * 2 == 12

    len(82) → Erreur

    len(['toto', 'tata', 'paille']) == 3

  • 7. La fonction range()

    La fonction range() est utilisée pour générer une séquence d'entiers dans un intervalle spécifié. Elle est souvent utilisée dans les "boucles for" pour itérer sur une séquence d'entiers.

    En Python, la fonction range() peut être utilisée de plusieurs manières, mais la forme la plus courante est range(n),
    où n est la valeur de fin de la séquence (non incluse):
    Par exemple, range(5) générera la séquence d'entiers de 0 à 4 soit 0, 1, 2, 3, 4.

    Une autre forme courante est range(n, p),
    où n est la valeur de départ de la séquence et p la valeur de fin (non incluse):
    Par exemple, range(2, 8) générera la séquence d'entiers de 2 à 7.
